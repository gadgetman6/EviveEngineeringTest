# Menu Ordering System

This program contains an object-oriented menu ordering system built to facilitate different combinations of items ordered at a restaurant during different meals of the day.

## Dependencies

This program requres a working version of Python (3.7.6 or greater) to be installed on the user's machine

## Usage

### Method 1:

1. Edit input file to the input desired to be read into the program. Note: only the first line of the input file will be processed.
2. Run following in command line:
```bash
python main.py < input.txt
```
3. Program output will be printed in command line.

### Method 2:
1. Run following in command line:
```bash
python main.py
```
2. Type input desired to be read into the program. Press enter to run the program on input.
3. Program output will be printed in command line.
